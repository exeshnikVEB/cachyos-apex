# CachyOS Plasma Apex 🌌

> Премиальная, красивая и функциональная сборка (конфигурация) KDE Plasma 6 для **CachyOS / Arch Linux**.

Этот репозиторий содержит скрипты для автоматического сохранения (экспорта) и установки (импорта) вашей конфигурации рабочего стола KDE Plasma 6.

---

## 🚀 Особенности сборки
- **Автоматизация:** Установка одной командой, включая системные и AUR-пакеты.
- **Безопасность:** Автоматическое создание бэкапа (`~/.config/kde-backup-<date>`) перед установкой, чтобы вы могли вернуться к своим старым настройкам в любой момент.
- **Гибкость:** Вы можете настраивать систему в графическом интерфейсе и экспортировать изменения обратно в проект одной командой.

---

## 📦 Быстрая установка

Чтобы установить эту сборку на чистую CachyOS с KDE Plasma 6:

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИКНЕЙМ/cachyos-apex.git
   cd cachyos-apex
   ```

2. Запустите скрипт установки:
   ```bash
   ./install.sh
   ```

---

## 🛠️ Как обновить сборку (Экспорт ваших настроек)

Если вы изменили внешний вид (темы, обои, панели, горячие клавиши) и хотите сохранить это в репозиторий, чтобы выложить на GitHub:

1. Запустите скрипт экспорта:
   ```bash
   ./capture.sh
   ```

2. Закоммитьте и отправьте изменения в ваш репозиторий:
   ```bash
   git add .
   git commit -m "update layout and themes"
   git push origin main
   ```

---

## 📁 Структура проекта
- `install.sh` — Скрипт установки сборки.
- `capture.sh` — Скрипт захвата ваших текущих настроек.
- `configs/` — Папка со всеми конфигурационными файлами KDE Plasma (`kdeglobals`, `plasmashellrc`, панели, виджеты и т.д.).
- `local-data/` — Темы, цветовые схемы, иконки и обои.
- `pkglist.txt` — Список установленных пакетов из официальных репозиториев.
- `aur-pkglist.txt` — Список установленных пакетов из AUR.
